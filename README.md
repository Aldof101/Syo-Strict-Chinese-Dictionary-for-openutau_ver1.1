Syo Strict Chinese Dictionary for openutau_ver1.0
by Aldof
This dictionary is based on Vivi's improved Syo Strict Chinese recording table (Reference: http://wiki.ivocaloid.com/information:tutorial:utau_109213). It allows Chinese phonetic sources using the Syo Strict method in OpenUtau to automatically split sounds by directly inputting Chinese characters or pinyin.
This yaml file references this repository: https://github.com/stakira/OpenUtau/pull/776

Usage:

1. Copy and paste the zhcvv.yaml file from the extracted folder into the directory of the OpenUtau singer using the Syo Strict Chinese scheme. For example, paste zhcvv.yaml into E:\OpenUtau\Singers\Heyu_Chinese.

2. In OpenUtau, select the "ZH CVV (October-style Extended Phoneme)" phonemizer.

3. Refresh the singer, and directly input Chinese lyrics or pinyin in the editing interface to automatically split sounds.

Syo厳式中国語辞書 for openutau_ver1.0
by Aldof
この辞書は、Viviの改良版Syo厳式録音表（参考: http://wiki.ivocaloid.com/information:tutorial:utau_109213）を基に作成されており、OpenUtauでSyo厳式中国語音源を使用する際、中国語文字またはピンインを直接入力することで自動的に音を分割できます。
このyamlファイルは、以下のリポジトリを参考にしています：https://github.com/stakira/OpenUtau/pull/776

使用方法：

1、解凍したフォルダ内のzhcvv.yamlを、OpenUtauでSyo厳式中国語スキームを使用する歌手のディレクトリにコピーして貼り付けます。例：zhcvv.yamlをE:\OpenUtau\Singers\Heyu_Chineseに貼り付けます。

2、OpenUtauで「ZH CVV（十月式整音拡張）」フォネマイザーを選択します。

3、歌手を更新し、編集インターフェースで中国語歌詞またはピンインを直接入力すると、自動的に音が分割されます。
